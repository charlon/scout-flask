# scout-hybrid
Flask rewrite of scout-hybrid application

## Getting Started
aksdjfaslkdf

### Prerequisites
* sdf
* virtualenv

### Installation

Clone this repository:
```
  $ git clone https://github.com/charlon/scout-hybrid.git
  $ cd scout-hybrid
```
Create/start Virtualenv:
```
  $ virtualenv .
  $ source bin/activate
```

Run Flask
```
  $ export FLASK_APP=run.py
  $ flask run
```
